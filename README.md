# Install
## 1 - composer update
### dump-autoload
## 2 - migrate datebase
### php artisan migrate 
### php artisan db:seed
## 3 - Generate migration from database
### php artisan ybr:gen-migrations
## 4 - Generate Models and CRUD from tablename
### php artisan ybr:gen-crud {tablename} {--force}
